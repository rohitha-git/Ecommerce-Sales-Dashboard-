# E-Commerce Sales & Customer Analytics Dashboard

## Project Overview
This project presents an end-to-end **E-Commerce Sales & Customer Analytics Dashboard** built using **Python, SQL, and Power BI**. The goal of this project is to analyze sales performance, customer purchasing behavior, and profitability trends to derive actionable business insights.

The dashboard enables interactive exploration of key business metrics such as total sales, profit, orders, customer count, category performance, and sub-category profitability.

---

## Objectives
- Analyze overall sales and profit performance
- Identify high-performing product categories
- Understand customer purchasing behavior
- Detect profitable and loss-making sub-categories
- Build an interactive business dashboard for business decision-making

---

## Tech Stack
- Python — Data preprocessing and cleaning  
- MySQL — SQL analysis and business queries  
- Power BI — Interactive dashboard visualization  
- Libraries: Pandas, NumPy, Matplotlib  

---

## Dataset
The dataset contains **9,994 retail transactions** with attributes such as:

- Order ID  
- Order Date  
- Customer Name  
- Region  
- Category  
- Sub-Category  
- Sales  
- Profit  
- Discount  
- Quantity  

---

## Project Workflow

### 1. Data Preprocessing (Python)
- Removed missing values
- Removed duplicate records
- Converted date columns into proper datetime format
- Created derived columns such as Order Month and Order Year
- Prepared clean dataset for SQL and Power BI analysis

### 2. SQL Analysis
Performed analytical SQL queries such as:
- Top 10 customers by sales
- Category-wise sales analysis
- Region-wise profit analysis
- Monthly sales trends
- Profitability analysis by sub-category

### 3. Dashboard Development (Power BI)
Built an interactive dashboard with:
- KPI Cards (Sales, Profit, Orders, Customers)
- Monthly Sales Trend Analysis
- Category-wise Sales Distribution
- Profit by Sub-Category
- Top Customer Analysis
- Dynamic Filters (Region, Category, Segment)

---

## Key Insights
- Technology category generated the highest sales revenue
- Copiers produced the highest profit among sub-categories
- Certain sub-categories such as Bookcases showed losses
- Sales peaked during year-end months
